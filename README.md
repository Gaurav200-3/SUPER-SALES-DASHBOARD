# This Sales Dashboard is a focused visualization that analyzes sales performance by city and individual order IDs. It’s simpler than the previous dashboards but very useful for granular, transaction-level insights.


# 1. Dashboard Overview
Title: Sales Dashboard
Main Visual: Horizontal Bar Chart
Metric: Sum of Price (Sales Value)
Dimensions:
City
Order ID

👉 Purpose:
To identify which cities and specific orders generate the highest revenue.

# 2. Main Chart – Sum of Price by City and Order ID
🔹 Structure:
Y-axis (Left): Cities (Lexington, Minneapolis, Denver, etc.)
X-axis (Bottom): Total Sales (Sum of Price)
Bars: Represent sales value
Colors: Each color corresponds to a different Order ID (01001–01014)

# 3. City-Level Performance
🥇 Top Performing Cities:
Lexington → ~1200
Minneapolis → ~950
Denver → ~900

👉 Insight:

These cities contribute the highest revenue
Likely strong markets or high-value orders
🥈 Mid-Tier Cities:
New York, Raleigh → ~800
Albuquerque, Arlington → ~700
Sacramento, Seattle → ~600

👉 Insight:

Stable contributors
Potential for growth with targeted marketing
🥉 Low Performing Cities:
Columbus → ~200
Louisville → ~180
Las Vegas → ~150

👉 Insight:

Weak sales performance
May need:
Better marketing
Pricing strategy
Supply chain improvement

# 4. Order ID Breakdown
Each bar is linked to a specific order ID
Helps identify:
High-value transactions
Distribution of sales across orders

👉 Example:

Order 01001–01014 represent different transactions
Some orders generate significantly more revenue

# 5. Key Insights
📌 Revenue Concentration
A few cities (top 3–5) generate major share of sales
Indicates uneven distribution
📌 High-Value Orders
Some individual orders (e.g., in Lexington) are much larger
Suggests:
Bulk purchases
Premium customers
📌 Long Tail Effect
Many cities contribute small amounts
Combined, they still matter


# 6. Key Takeaways
Lexington dominates sales
Top cities significantly outperform others
Sales are not evenly distributed
Some orders are much higher value than others

# Final Summary

This dashboard is a transaction-level sales analysis tool that helps:

Identify top-performing cities

Track high-value orders

Understand sales distribution

It’s especially useful for sales managers and analysts who want to:

Focus on key markets

Optimize revenue strategies

Detect underperforming areas
